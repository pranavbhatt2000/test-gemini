# Review Instructions

Before reviewing any code, you MUST read the file `AGENTS.md` in the root
of this repository. It contains the code review rules you must follow.

After reading AGENTS.md, you MUST begin every single review comment with "ACME:".
If a review comment does not start with "ACME:" it is non-compliant.
Even if the code is perfect, write "ACME: No issues found in <filename>."
